## 📖 Project Overview

This project is a responsive Temperature Converter developed using **HTML5, CSS3, and JavaScript**. It allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin with accurate calculations and an intuitive user interface.

---

## ✨ Features

- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Responsive and modern user interface
- Input validation
- Instant conversion results

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objective

The aim of this project is to strengthen JavaScript programming skills by implementing temperature conversion logic while designing a clean, responsive, and interactive web application.
